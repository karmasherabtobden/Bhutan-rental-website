# Bhutan Rental Website

A simple website that connects tenants and landlords in Bhutan.

## Features (MVP)
- User registration (Tenant / Landlord)
- Post rental listings
- Search rental listings
- Contact property owner

## Tech Stack
- HTML, CSS, JavaScript
- Firebase (Auth, Firestore, Storage)
- Firebase Hosting
